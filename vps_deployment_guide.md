## 🛠 Phase 1: Initial Server Setup

Login to your VPS via SSH:
```bash
ssh root@your_vps_ip
```

Install the foundation:
```bash
# Update system
sudo apt update && sudo apt upgrade -y

# Install Python, Node.js, Nginx, and Git
sudo apt install -y nginx git python3-pip python3-venv nodejs npm

# Install PM2 globally (Process Manager to keep app running)
sudo npm install -g pm2
```

## ⚙️ Phase 3: Backend Deployment (FastAPI)

1. **Clone Repo**:
```bash
cd /var/www
git clone https://github.com/suleiman-code/Digital-Points.git
cd Digital-Points/backend
```

2. **Environment Setup**:
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

3. **Configure Environment Variables**:
```bash
nano .env
```
Paste your production settings:
```env
MONGODB_URL=mongodb+srv://user:pass@cluster.mongodb.net/digitalpoint
RESEND_API_KEY=re_your_api_key
MAIL_FROM=info@yourdomain.com
MAIL_FROM_NAME=Digital Points Admin
ADMIN_CONTACT_EMAIL=your@email.com
SECRET_KEY=generate_a_random_string
```

4. **Start Backend with PM2**:
```bash
pm2 start "uvicorn main:app --host 0.0.0.0 --port 8000" --name "digital-backend"
```

## 🌐 Phase 5: Nginx Reverse Proxy

Nginx will route traffic from Port 80 (Internet) to your apps.

1. **Create Config**:
```bash
sudo nano /etc/nginx/sites-available/digitalpoints
```

2. **Paste Config**:
```nginx
server {
    listen 80;
    server_name yourdomain.com www.yourdomain.com;

    # Frontend
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }

    # Backend API
    location /api {
        proxy_pass http://localhost:8000/api;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

3. **Enable & Restart**:
```bash
sudo ln -s /etc/nginx/sites-available/digitalpoints /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```
